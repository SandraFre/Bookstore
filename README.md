#Bookstore application

GIT
https://github.com/SandraFre/Bookstore

**API Using**
* Java 16
* Spring Boot
* Hibernate
* SQL (H2, Postgres)
* Spring Security
* Thymeleaf
* Lombok

**UI Using**
* Npm
* React.js
* Mui

**To start**
* Run **BookstoreApiApplication** on API side
* Run **npm install** (only if running for the first time) and **npm start** on UI side.

**Users:**
* admin (psw: admin)
* user (psw: user)
